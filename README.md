## Marco Mesen WebSite - Blog and Portfolio
### [`https://marcomesen.com/`](https://marcomesen.com/)

This WebSite was build with React.js, Hooks, Material UI, Firebase and Next.js

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
